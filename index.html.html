    Email Service Status ⚠ Not Configured

    RITUAL uses EmailJS to send real OTP verification emails — free, no backend required, works directly from the browser.
    Free tier: 200 emails/month.

    Setup Guide3 steps · 5 minutes
    1
    Create a free EmailJS account
    Go to emailjs.com → Sign up free → Go to Email Services → Connect Gmail, Outlook or any SMTP provider → copy your Service ID.
    2
    Create an Email Template
    Go to Email Templates → Create Template → Use these exact variables in your template body:
    To: {{to_email}}
    Subject: Your RITUAL verification code

    Hi {{user_name}},

    Your verification code is: {{otp_code}}

    This code expires in 10 minutes.
    If you didn't request this, ignore this email.     Copy your Template ID.
    3
    Get your Public Key
    Go to Account → API Keys → copy your Public Key. Paste all three values below and save.
    EmailJS Credentials
    Account → API Keys → Public Key
    Email Services → your connected service
    Email Templates → your OTP template
